# 手描きパターンデザイナー

プロのイラストレーターによる手描きパターンを忠実に再現し、パラメトリックに変化させることができるWebアプリケーション。

## 機能

- 9種類の手描きパターン
  1. 縦波線
  2. カンマ散布
  3. 手描き円
  4. リング散布
  5. スターバースト
  6. 重なる四角
  7. 白樺ダッシュ
  8. スクリブル
  9. 横波線

- パラメータ調整
  - 有機的な揺らぎ
  - 線の太さ
  - 密度
  - スケール
  - 回転

- シード値による再現性
- SVG/PNG出力

## 使い方

`tegaki-pattern-designer.html` をブラウザで開くだけで動作します。

## ファイル構成

- `tegaki-pattern-designer.html` - メインアプリケーション（最新版）
- `parametric-pattern-designer.html` - 旧バージョン
